The Open Pedalboard Mainboard is the central PCB of the modular pedalboard system.
It provides power distribution, MIDI routing, and audio switching for up to 8 effect pedals.

## Features

- 8 effect loops with relay-based true bypass
- MIDI over USB and DIN
- 9V and 12V regulated power outputs
- Raspberry Pi Compute Module 4 host
- Designed with [KiCad](https://www.kicad.org/)
- [OSHWA certified (CH000023)](https://certification.oshwa.org/ch000023.html)
