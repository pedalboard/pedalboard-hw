Mainboard for processing audio and/or MIDI data on a pedalboard.
Modular design supporting MIDI-only or combined MIDI + audio configurations.

## Key Features

- Raspberry Pi CM4/CM5 for audio processing (< 2ms latency with ELK Audio OS)
- RP-2040 MIDI controller with independent firmware and instant startup
- 6 foot buttons (short/long press, push/release) + 2 rotary encoders
- 8 RGB LED ring connectors + 2 OLED display ports
- Flexible MIDI I/O: DIN 5, 3.5mm TRS, USB MIDI, header pins
- 2 expression pedal inputs
- USB-A host + Mini USB device (firmware update)
- 6–28V DC input with 3A buck converter, or USB powered
- [OSHWA certified (CH000023)](https://certification.oshwa.org/ch000023.html)
